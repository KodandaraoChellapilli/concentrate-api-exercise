# Concentrate AI API Exercise

This project tests the Concentrate AI API using two providers: OpenAI and Anthropic. It loops through multiple questions and temperatures, prints responses, and saves results to a CSV file.

Note: The script now hits the real API endpoint. If your account has insufficient credits, you may see Error 402 responses. The script works fully with real API calls when credits are available.

Files

script.py – Main Python script

results.csv – CSV file with results

.env – Environment file for API key and URL (keep your key secret!)

How to Run

Create a .env file with your API key and URL:

API_KEY=your_concentrate_api_key
API_URL=https://api.concentrate.ai/v1/responses

Install dependencies:

pip install requests pandas python-dotenv

Run the script:

python script.py

Open results.csv to see the output.

Features

Loops through multiple questions

Uses OpenAI and Anthropic models

Tests different temperature settings

Saves results to CSV

Simple, easy-to-read Vinny-style code
